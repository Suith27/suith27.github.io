NEUQ 1010: C基础-阶乘数列

发表于 2016-08-29

题目链接

题目大意

输出：1+2!+3!+4!+…+30!。
格式：科学计数法，保留两位小数。

思路

unsigned long long 都会炸掉，别问我怎么知道的。
float 却不会，毕竟伸缩自如，只留总要的头部和长度，记不清的可以补一补浮点型数据如何储存的。
用一个 fo r循环就可以解决，毕竟 (n-1)! 跟 n! 只差一个 n。
至于输出格式，C语言可以用 %e,C++可以用 setprecision(2) 和 scientific。

PS

答案是2.74e+32，直接输出应该也是可以的。
又刷了一边1001-1005的a+b基础题，感觉自己有点弱智，还是不放上来代码了，要不然太水，虽然这道也挺水的。
